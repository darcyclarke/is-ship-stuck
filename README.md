# is-ship-stuck
Is the ship stuck still?

### Usage

#### CLI

```bash
npx is-ship-stuck
```

#### Dependency

```bash
npm i is-ship-stuck
```

```js
const isShipStuck = require('is-ship-stuck')()
```

Returns a boolean whether or not the ship is stuck (ref. https://istheshipstillstuck.com/)...
